This example is a dropin integration with advanced flow. For more information on how to configure this integration please refer to the documentation.

With Adyen you can do ThreeDS2 Authentication two ways; native or redirect. In a native flow, the authentication occurs in the users browser via the Web SDK. In a redirect flow, the user is redirected to an Adyen domain to complete the authentication.
